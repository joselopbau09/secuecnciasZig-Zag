## Ejecución
Para ejecutar el programa se necesita:
- Python 3

Se descomprime el archivo de tipo ".zip" luego:
1. Dirigirse en la terminal a la carpeta src: 
```
JoseLuisLopez\src$
```
2. Ejecutar el comando 
```
Python3 main.py
```
